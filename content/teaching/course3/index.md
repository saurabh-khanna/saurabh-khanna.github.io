## Course Description

This advanced course introduces students to state-of-the-art computational methods for communication research. Building on foundational skills, students learn to apply advanced machine learning and natural language processing techniques to analyze large-scale communication data. Topics include deep learning, transformer models, network analysis, and ethical considerations in computational research.
